# 🌍 IP Address Lookup

A Python tool that converts a website domain into its IP address.

## Features

- Resolve domain names
- Display IPv4 addresses
- Fast and simple

## Requirements

- Python 3.x

## How to Run

```bash
python ip_lookup.py
```

## Example

```
Enter website:
google.com

IP Address:
142.250.xxx.xxx
```
